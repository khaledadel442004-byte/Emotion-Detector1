# 🚨 Motion Detector System (Smart Security)

هذا المشروع عبارة عن نظام إنذار ذكي يعتمد على الأردوينو لاكتشاف الحركة وتنبيه المستخدم فوراً.

## 📝 وصف المشروع (Description)
نظام أمان بسيط وفعال يستخدم حساس الحركة **PIR** لمراقبة المكان. بمجرد اكتشاف أي حركة في النطاق، يقوم النظام بـ:
1. إطلاق إنذار صوتي عبر الـ **Buzzer**.
2. عرض رسالة تحذيرية "HEY! I SEE YOU!" على شاشة الـ **LCD**.
3. العودة لوضع المسح (Scanning) تلقائياً عند استقرار المكان.

## 🛠️ المكونات (Hardware Required)
* **Arduino Uno** (أو أي لوحة متوافقة)
* **PIR Motion Sensor** (حساس الحركة)
* **LCD I2C 16x2** (شاشة عرض)
* **Active Buzzer** (جرس إنذار)
* **Jumper Wires** (أسلاك توصيل)

## 🔌 التوصيلات (Wiring)
| المكون | منفذ الأردوينو (Pin) |
| :--- | :--- |
| **PIR Sensor** | Pin 3 |
| **Buzzer** | Pin 9 |
| **LCD SDA** | A4 |
| **LCD SCL** | A5 |

## 🚀 كيفية التشغيل
1. قم بتحميل مكتبة `LiquidCrystal_I2C`.
2. ارفع الكود الموجود في ملف `Motion_Detector.ino` إلى لوحة الأردوينو.
3. انتظر 30 ثانية لمعايرة الحساس (Calibration).
4. استمتع بنظام الأمان الخاص بك!
